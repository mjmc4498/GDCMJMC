Manual del Sistema

Este sistema ha sido desarrollado para facilitar el aprendizaje de los dominios DAMA-DMBOK.

Estructura del Proyecto

  • index.html: Dashboard principal.
  • css/styles.css: Estilos globales (Basado en CSS3).
  • js/app.js: Lógica de navegación y persistencia (LocalStorage).
  • modules/: Contenido educativo específico.

Tecnologías Utilizadas

HTML5, CSS3 (Modern UI), JavaScript (Vanilla).
